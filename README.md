# LEZECK STORE - E-Commerce Website

A fully functional e-commerce website with shopping cart, user accounts, payment integration, and responsive design.

## Features

- 🛍️ Product catalog with 30+ products
- 🛒 Shopping cart with quantity management
- 💳 Payment simulation (Bank Transfer, Alipay, WeChat Pay)
- 👤 User registration and login system
- 📋 Order history
- 🔍 Product search and filtering
- 💰 Price range filter
- 📱 Fully responsive design
- ✨ Fly-to-cart animation

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (Vanilla, no frameworks)
- LocalStorage for data persistence
